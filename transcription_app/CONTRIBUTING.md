# Voice to Text Project 
An audio transcription application. 

⚡ Features:
- voice to text audio transcription with Whisper AI
- key point detection
- speaker identification / diarization using pyannote-audio
- basic UI


👨‍💻 Tech Used: 
 Python
 FastAPI
 Whisper AI


## Run Locally
```
poetry run python app.py
```
